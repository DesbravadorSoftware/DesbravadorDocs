# Pinpads Homologados

> ℹ️ Esta é uma página compartilhada entre todos os sistemas Desbravador. Última atualização: 24/05/2026.

---

Pinpads são os periféricos de captura de senha conectados às estações de trabalho do PDV. Os modelos abaixo são homologados para uso com os sistemas de TEF integrados ao Desbravador.

> ⚠️ **Atenção** — Modelos com versão de firmware especificada requerem exatamente aquela versão para correto funcionamento. Verificar a versão instalada antes da implantação.

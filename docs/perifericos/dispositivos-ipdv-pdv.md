## Observações Gerais

- A homologação de dispositivos é específica por versão do sistema Desbravador e versão do aplicativo iPDV. Atualizações podem ampliar ou alterar a lista de dispositivos suportados.
- Dispositivos marcados como descontinuados não receberão novas atualizações de integração. Recomenda-se migração para modelos ativos antes do fim do suporte.
- Para uso de NFC com pagamento por aproximação, verificar se o modelo possui chip NFC ativo e se o adquirente suporta essa modalidade na integração contratada.
- Dispositivos utilizados como terminais de autoatendimento (totens) podem ter requisitos adicionais de configuração — consultar a equipe Desbravador.
